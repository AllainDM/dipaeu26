## Структура проекта

- `turns.js` — массив `turnsData` с объектами ходов
- `trade-data.js` — объекты `costData` и `countData` (ключи — номера ходов)
- `trade.html` — SPA с навигацией, внутри скрипт с массивом `turns`
- `log.html` — лог изменений, новая запись вставляется после `<p>История изменений:</p>`
- `map-script.js` — константа `LATEST_POLIT_MAP`

## Формат turns.js

```javascript
{
    id: N,
    title: "N-й ход (ГГГГ г.)",
    map: "maps/polit/N.png",
    trade: "turnN/trade.html",
    content: `<p><b>Событие</b></p>
    <p><b>Описание [механики]</b></p>
    <p>· Событие 1.</p>
    <p>· Событие 2.</p>
    <p>· Основаны новые города: Город (Страна)</p>`
}
```

## Порядок действий при добавлении хода

1. Запросить у пользователя: номер хода, год, текст событий
2. Запросить данные торговли (цены и/или количество по 8 ТЦ × 36 товаров) — если есть
3. Запросить: есть ли новая политическая карта, есть ли торговые изображения
4. Добавить объект хода в `turns.js` (последним, перед `];`)
5. Если есть данные цен — добавить в `costData` объекта `trade-data.js`
6. Если есть данные количества — добавить в `countData` объекта `trade-data.js`
7. Если ход > 31 — добавить объект в массив `turns` внутри `trade.html`
8. Создать `turns/turn<id>/trade.html` с минимальной разметкой
9. Обновить `LATEST_POLIT_MAP` в `map-script.js` (если есть карта)
10. Вставить запись в `log.html` после `<p>История изменений:</p>`
11. Сделать коммит с сообщением на русском
