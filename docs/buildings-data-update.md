# Обновление данных построек

Данные для интерактивной таблицы (`buildings.html`) берутся из `dev/buildings.xlsx`.

После изменения xlsx-файла нужно перегенерировать `buildings.js`:

```bash
python3 dev/xlsx2js.py
```

Скрипт читает 11 колонок xlsx (заголовки: Категория — Доход) и создаёт `buildings.js` с массивом `BUILDINGS` и всей интерактивной логикой (фильтры, сортировка, цвета).

**Зависимость:** `python3` + `openpyxl`.
